#CS210-Corner-Grocer
C++ and Python item-tracking for "Corner Grocer" Programming languages class. 


#Project Summary

This project is a grocery item-tracking program for Corner Grocer. The program reads a list of items purchased during the day and counts how many times each item appears. It lets the user search for an item, view all item frequencies, and see a simple text-based histogram. This helps the store understand which items are bought most often.

#Reflection

One area I could improve is input validation and error handling. Adding more checks for invalid menu choices or missing files would make the program more reliable and user-friendly.

The most challenging part was working with both C++ and Python together. I handled this by testing each part carefully and using course resources and examples to troubleshoot problems.

The skills from this project that will help me in the future include file handling, using functions, writing readable code, and working with more than one programming language.
